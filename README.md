## README.md
Frontend for modulino web.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/modulino-lab/www
   ```
2. Install dependencies:
   > `npm` works just fine, but `pnpm` is recommended for better performance.
   ```bash
   cd www
   curl -fsSL https://get.pnpm.io/install.sh | sh - # if pnpm is not installed
   pnpm install
   ```
3. Start the development server:
   ```bash
   pnpm dev
   ```
